# 🤖 企业文档知识库问答机器人

## 项目简介
基于 RAG（检索增强生成）技术和通义千问大模型的企业级文档问答系统，支持上传 PDF 格式的内部文档（如员工手册、产品说明书、制度规范等），通过自然语言快速检索并精准回答文档内容，帮助企业提升内部知识获取效率。

## 技术栈
| 模块         | 技术/工具                     | 说明                     |
|--------------|------------------------------|--------------------------|
| 后端框架     | Python + FastAPI             | 提供 API 服务            |
| 向量检索     | NumPy 余弦相似度计算         | 轻量级本地检索（可扩展） |
| 大模型服务   | 通义千问（Qwen-turbo）        | 生成式问答能力           |
| 前端界面     | Streamlit                    | 快速搭建交互页面         |
| 文档解析     | pdfplumber                   | PDF 文本提取             |
| 环境管理     | python-dotenv                | 安全存储 API Key         |
